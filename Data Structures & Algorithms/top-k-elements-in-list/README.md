# Top K Frequent Elements - LeetCode 347

## Problem Description

Given an integer array nums and integer k, return the k most frequent elements.

## Examples

```
Input: nums = [1,1,1,2,2,3], k = 2
Output: [1,2]
```

## Topics

- Array
- Hash Table
- Sort
- Heap (Priority Queue)
- Bucket Sort

## Solution: Bucket Sort / Heap
